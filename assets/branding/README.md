# IndusMind AI — Brand Guidelines

## Brand Identity

**Name**: IndusMind AI
**Tagline**: "The AI Brain for Industrial Operations"

## Logo

The IndusMind AI logo is a stylized brain-gear fusion icon representing the intersection of artificial intelligence and industrial operations.

- Use `assets/logo.png` for the logo icon
- Use `assets/banner.png` for the hero banner

## Color Palette

### Primary

| Color | Hex | Usage |
|-------|-----|-------|
| Indigo 600 | `#4f46e5` | Primary actions, links |
| Indigo 500 | `#6366f1` | Hover states, gradients |
| Violet 600 | `#7c3aed` | Secondary gradient stop |
| Violet 500 | `#8b5cf6` | Accent elements |

### Semantic

| Color | Hex | Usage |
|-------|-----|-------|
| Emerald 400 | `#34d399` | Success, positive trends |
| Amber 400 | `#fbbf24` | Warnings, caution |
| Red 500 | `#ef4444` | Errors, critical alerts |
| Cyan 500 | `#06b6d4` | Information, equipment |

### Background

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#09090b` | Page background |
| Card | `#18181b` | Card surfaces |
| Muted | `#27272a` | Input fields, secondary surfaces |
| Border | `#27272a` | Borders, dividers |

## Typography

- **Headings**: System font stack, `font-weight: 700`
- **Body**: System font stack, `font-weight: 400`
- **Monospace**: `ui-monospace, SFMono-Regular, Menlo`

## Design Principles

1. **Dark-first**: All designs start from dark mode
2. **Glassmorphism**: Cards with backdrop blur and subtle borders
3. **Gradient accents**: Indigo-to-violet gradients for CTAs and active states
4. **Micro-animations**: Subtle hover effects and page transitions
5. **Information density**: Dense but readable dashboard layouts
