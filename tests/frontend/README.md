# Frontend Tests

This directory will contain the test suite for the IndusMind AI frontend.

## Planned Setup

- **Test Runner**: Vitest or Jest
- **Component Testing**: React Testing Library
- **E2E Testing**: Playwright (future)

## Running Tests (Once Configured)

```bash
cd frontend

# Run tests
npm test

# Run with coverage
npm run test:coverage

# Run E2E tests
npx playwright test
```
